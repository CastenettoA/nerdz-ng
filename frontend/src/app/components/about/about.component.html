About Nerdz Ng

Nerdz Ng è un porting in Angular del social network opensource chiamato Nerdz.

Nerdz Ng è un progetto open source ed il suo codice sorgente è disponibile su Github distribuito con licenza GPL-3.0.

E' un social network basato sulle Api pubbliche del progetto Nerdz.eu.

Se vuoi aiutare lo sviluppo e il mantenimento di Nerdz Ng puoi diventare un sponsor, oppure, essendo un progetto open source puoi collaborare direttamente tramite code review, bug fixing e sviluppo di nuove funzionalità.

Puoi segnalare un bug o proporre una miglioria funzionale e/o grafica aprendo una issue su Github.
